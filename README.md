
# Hoo_Bank

Hoobank is a user-friendly and secure frontend application for managing and visualizing online payments. Designed with React.js, it offers a seamless interface for handling various payment processes.



## Technologies Used


- HTML
- tailwind css
- JavaScript
- React.js
